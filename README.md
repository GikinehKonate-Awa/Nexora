# NEXORA CONSULTING GROUP - Sistema de Fichaje

Aplicación web de control de presencia desarrollada 100% con tecnologías nativas: HTML5, CSS3, JavaScript Vanilla, PHP y MySQL. Sin frameworks ni librerías externas de ningún tipo.

## ✅ Características

- ✅ Detección automática de VPN corporativa para fichaje
- ✅ Soporte para modalidad teletrabajo
- ✅ Desfichaje nocturno manual con comentario obligatorio
- ✅ Roles: Empleado, Jefe de Departamento, Admin / RRHH
- ✅ Sistema de notificaciones y alertas
- ✅ Gestión de horas extras y solicitudes
- ✅ Reportes y exportación CSV/PDF
- ✅ Diseño responsive premium corporativo

## 🛠️ Requisitos Técnicos

- PHP 7.4+
- MySQL 5.7+
- Extensiones PHP: PDO, pdo_mysql, gd
- Servidor web Apache / Nginx

## 🚀 Instalación

1. Clonar repositorio
2. Acceder a `/install.php` desde el navegador
3. Seguir los pasos de instalación automática
